# TypeScript WebGPU Renderer

**Experience it here: https://jtsorlinis.github.io/js-renderer/**

This branch is WebGPU-only. It renders a normal-mapped OBJ model in the browser using WGSL shaders and GPU pipelines.

## Features
- WebGPU rendering pipeline
- Perspective and orthographic projection
- Wireframe and shaded material modes
- Texture and normal mapping
- Shadow mapping
- OBJ model loading

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
